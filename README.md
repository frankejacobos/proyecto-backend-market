# proyecto-backend-martket
backend, api, nodejs, express, database, market. (nodev: 16.15.0), (npmv: 8.13.2).
el proyecto cuenta con:
- productos
  + categorias
- clientes
- distribuidores
- compras
- ventas

crear el archivo .env con las siguientes variables:
  - NODE_ENV=production
  - ROOT=exampleroot
  - PASSWORD=examplepassword
  - db=exampledatabase
